You can upload a TAR file and extract its contents.

  1. Create a TAR file. For example: 
    echo "Hello world" > hello.txt
tar cvf test.tar hello.txt
  2. Upload it to this site. You will receive a UUID. (For example: fe8fd755-f1d3-4648-af91-032b6ff1e011)
  3. You can access the extracted files at /static/<UUID>/<file name> (for example, /static/fe8fd755-f1d3-4648-af91-032b6ff1e011/hello.txt).
